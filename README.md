### Real-Time ICU Patient Monitoring Platform
> Azure • Microsoft Fabric • Event Hub • ADLS Gen2 • Power BI • Python

End-to-end real-time healthcare data platform built on **Medallion Architecture** (Bronze/Silver/Gold) with live ICU patient vital signs streaming, clinical risk scoring, and automated daily pipeline.

- Power BI dashboard with conditional clinical threshold formatting
- Delta Lake tables via Microsoft Fabric Lakehouse
